## 特色的
```
©	&copy;	&#169;	版权标志
|	 	&#124;	竖线，常用作菜单或导航中的分隔符
·	&middot;	&#183;	圆点，有时被用来作为菜单分隔符
↑	&uarr;	&#8593;	上箭头，常用作网页“返回页面顶部”标识
€	&euro;	&#8364;	欧元标识
²	&sup2;	&#178;	上标 2，数学中的平方，在数字处理中常用到，例如：1000²
½	&frac12;	&#189;	二分之一
♥	&hearts;	&#9829;	心型，用来表达你的心
```
