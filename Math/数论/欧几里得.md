# 欧几里得算法/辗转相除法

## 1. 算法定义/流程

- 算法定义

  两个整数的最大公约数 (greatest common divisor, gcd) 等于其中较小的那个数和两数相除余数的最大公约数：
  ```
  gcd(a, b) = gcd(b, a mod b)
  ```

- 算法流程

  ```
  设
    a = 51, b = 36
  则：
    51 % 36 = 15
    36 % 15 = 6
    15 % 6 = 3
    6 % 3 = 0
  因此：
    gcd(a, b) = 6
  ```
